## Required File Structure

You can also simply export the deck to see an example of the file format.

| Column | Required | Data Type | Description |
|--------|----------|-----------|-------------|
| **Term** | ✅ Required | Text | Question/prompt (front of card) |
| **Definition** | ✅ Required | Text | Answer (back of card) |
| **Disabled** | ❌ Optional | Boolean | TRUE/FALSE - card enabled status |

**Important:**
- Order doesn't matter.

## CSV Format (.csv)

```csv
Term,Definition,Disabled
"What is DNA?","Deoxyribonucleic acid",FALSE
"Capital of France?","Paris",FALSE
```
