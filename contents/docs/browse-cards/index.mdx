## Browse Mode vs. Study Mode

Browse Mode is a **view-only** version of Study Mode:

| Feature | Study Mode | Browse Mode |
|---------|-----------|-------------|
| **Cards shown** | Only due cards | All cards in deck |
| **Grading buttons** | ✅ Yes (Easy, Good, Hard, Again) | ❌ No |
| **View term & definition** | ✅ Yes | ✅ Yes |
| **Swipe between cards** | ✅ Yes | ✅ Yes |
| **Edit/delete cards** | ✅ Yes | ✅ Yes |
| **Interactive gestures** | ✅ Yes | ✅ Yes |

**Use Browse Mode when you want to:**
- Review all cards without studying
- Preview a deck's content
- Check what's in a deck without changing progress

---

## Starting Browse Mode

From the home screen:

**Tap a deck** → **"Browse Cards"**

All cards appear in carousel view.
