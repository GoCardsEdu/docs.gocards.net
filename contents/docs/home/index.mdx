## Creating a Deck

1. **Tap the "+" button** (bottom-right, available in Recent Decks or All Decks tabs)
2. **Enter deck name** (e.g., "Spanish Verbs", "Biology Chapter 3")
3. **Tap "Create"**

The deck appears in your All Decks list and will show in Recent Decks once you use it.

---

## Creating a Folder (Premium)

Organize related decks into folders.

1. **Go to "All Decks" tab**
2. **Tap "‚ãÆ" menu** (top-right)
3. **Select "New Folder"**
4. **Enter folder name** (e.g., "Spanish", "Medical School", "Work Training")
5. **Tap "Create"**

You can create folders inside folders for hierarchical organization.

---

## Deck Menu Actions

**Tap any deck** to see these options:

### Study & Review
- **Study Cards** - Learn with spaced repetition
- **Browse Cards** (Premium) - View-only carousel (no grading)
- **List Cards** - See all cards in a list/grid

### Card Management
- **Add Card** - Create a new card quickly

### Organization
- **Cut Deck** - Move deck to a different folder
- **Rename Deck** - Change the deck name
- **Delete Deck** - Remove deck and all its cards (with confirmation)

### Advanced (More "‚ãÆ" menu)
- **Sync with file** (Premium) - Connect to Excel/CSV file
- **Export to Excel** (Premium) - Save deck as .xlsx
- **Export to CSV** (Premium) - Save deck as .csv
- **Export Database** - Backup deck as .db file
- **Deck Settings** - Configure auto-sync, forgotten card limits, etc.

---

## Folder Menu Actions

**Long-press any folder** to see:

- **Rename Folder** - Change folder name
- **Delete Folder** - Remove folder (must be empty first)
- **Cut Folder** - Move folder to a different location

---

## Searching Decks

In the **All Decks** tab:

1. **Tap the üîç search icon** (top-right)
2. **Type deck name** (search is case-insensitive)
3. **Results appear instantly** as you type
4. **Tap a result** to open that deck's menu

Search looks through deck names across all folders.

---

## Moving Decks Between Folders

To reorganize your deck structure:

1. **Long-press a deck**
2. **Select "Cut Deck"**
3. **Navigate to the destination folder** (or stay in root)
4. **Long-press the folder** (or empty space)
5. **Select "Paste Deck"**

The deck moves to the new location. The same process works for folders.

## Tips for Organization

- **Use folders by topic** - Group decks by subject, class, or project
- **Keep folder names short** - Easier to navigate on mobile
- **Recent Decks tab** - Your most-used decks are always one tap away
- **Export regularly** - Back up important decks (see [Database Backup](/docs/home/database-backup/))
- **Search is fast** - Don't over-organize; search works great for finding decks
