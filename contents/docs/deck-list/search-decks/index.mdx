---
title: Searching Decks and Folders (Premium feature)
description: Learn how to search for flashcard decks by name across all folders in GoCards.
keywords: ["search decks","deck search"]
---

In the **All Decks** tab:

1. **Tap the 🔍 search icon** (top-right)
2. **Type deck name** (search is case-insensitive)
3. **Results appear instantly** as you type
4. **Tap a result** to open that deck's menu

Search looks through deck names across all folders.
