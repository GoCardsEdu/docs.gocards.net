## Topbar Actions

On the topbar, you can see the following actions:
- **🔍 [Search](/docs/deck-list/search-decks)** (Premium) - Find decks by name across all folders
- **"+" button** to [create a new deck](/docs/deck-list/create-folder)
- Tap "⋮" in the topbar in the top-right to open the menu with these options:
    - [New Folder](/docs/deck-list/create-folder) (Premium)
    - [Import Excel](/docs/sync-export-import/import-cards)
    - [Import CSV](/docs/sync-export-import/import-cards)
    - [Import DB](/docs/deck-list/database-backup)
    - [Settings](/docs/settings/app)

## Deck Menu Actions

**Tap "⋮" on the deck** to open the menu with these options:
- [**Browse**](/docs/browse-cards) (Premium) - View-only carousel (no grading)
- [**List Cards**](/docs/cards-list) - See all cards in a list
- **Add a new card** - Create a new card quickly at the end
- [**Cut**](/docs/cards-list/reorder-cards) - Move deck to a different folder
- **Rename** - Change the deck name
- [**Delete**](/docs/cards-list/delete-cards) - Remove deck and all its cards (with confirmation)

<center><img src="/images/docs/deck-popup-menu.png" width="300" /></center>

Tap **More** in the deck menu to access:
- [**Sync with Excel/CSV**](/docs/sync-export-import/sync-cards) - Connect to an existing Excel/CSV file
- [**Export to new Excel**](/docs/sync-export-import/export-cards) - Save deck as .xlsx
- [**Export to new CSV**](/docs/sync-export-import/export-cards) - Save deck as .csv
- [**Export DB**](/docs/deck-list/database-backup) - Backup deck as .db file
- [**Deck Settings**](/docs/settings/deck-specific) - Configure auto-sync, forgotten card limits, etc.

## Folder Menu Actions

**Tap "⋮" on the folder** to open the menu with these options:
- **Rename** - Change folder name
- **Delete** - Remove folder with all desks (with confirmation)
- **Cut** - Move folder to a different location
