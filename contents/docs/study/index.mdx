## Starting a Study Session

GoCards automatically selects which cards to show based on:
- Cards you've never seen before
- Cards due for review today
- Cards you marked as "Again" recently

---

## Navigation

- **Swipe left/right** - Move between cards
- **Tap bottom section** - Reveal the definition (answer) and grading buttons

---

## Card Layout

```
┌─────────────────────────────┐
│    [Toolbar: ↶ + ✏️ 🗑️ ⋮]   │
├─────────────────────────────┤
│                             │
│        TERM (Question)      │ ← Top section (always visible)
│                             │
├─────────────────────────────┤ ← Drag this bar to resize
│                             │
│     DEFINITION (Answer)     │ ← Bottom section (tap to reveal)
│                             │
├─────────────────────────────┤
│  [Again] [Quick] [Hard] [Easy] │ ← Grading buttons
│   2 days   5 min   3 days  5 days│ ← Next review time
└─────────────────────────────┘
```

## Toolbar Options (Top)

- [**✏️ Edit Card**](/docs/cards-list/editing/)
- **⋮ More Menu**
    - **+ New** - Create new card
    - **🗑️ Delete** - Removes the current card (with confirmation)
