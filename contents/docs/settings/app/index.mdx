## Available Settings

### Dark Mode

**What it does:** Changes app color scheme.

**Options:**
- **Auto** (Default) - Follows Android system dark mode setting
- **Light** - Always light theme
- **Dark** - Always dark theme

---

## How to Access

**Home screen** → **"⋮" menu** (top-right) → **"Settings"**

Opens Settings screen with 3 tabs. This page covers tabs 2 and 3.
