GoCards offers flexible settings at three levels: deck-specific, all-decks, and app-wide. Access settings through the deck menu or app menu.

## How to Access Settings

### Deck Settings

1. **Tap on a deck** → **"More"** → **"Deck Settings"**
2. Or from **"List Cards"** → **"⋮" menu** → **"Deck Settings"** (gear icon)

### App Settings

1. **Home screen** → **"⋮" menu** (top-right) → **"App Settings"**

## Settings Organization

The Settings screen has **3 tabs:**

### Tab 1: This Deck
Settings for the current deck only. See [Deck Settings](/docs/settings/deck-settings/).

### Tab 2: All Decks
Default settings applied to all decks. See [App Settings](/docs/settings/app-settings/) - All Decks section.

### Tab 3: App
Global app preferences (dark mode, edge bars). See [App Settings](/docs/settings/app-settings/).

## Settings Hierarchy

**Priority order:**
1. **Deck-specific settings** (highest priority)
2. **All Decks defaults**
3. **App settings** (global)

**Example:** If "Limit forgotten cards" is set to 10 in "All Decks" but 5 in "This Deck", the deck uses 5.

## Common Settings Tasks

### Change Dark Mode
**App tab** → **Dark Mode** → Choose: Auto / Light / Dark

### Adjust Study Session Length
**This Deck tab** → **Limit forgotten cards** → Set number (e.g., 5, 10, 20)

### Enable Auto-Sync (Premium)
**This Deck tab** → **Auto-Sync** → Toggle ON

### Hide Edge Bars
**All Decks tab** → **Show left edge bar** → Toggle OFF
**All Decks tab** → **Show right edge bar** → Toggle OFF

### Change Card Display Lines
**This Deck tab** → **Max lines** → Set number (1-10)
