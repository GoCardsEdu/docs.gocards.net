Multi-select mode lets you perform bulk operations on multiple cards at once - perfect for reorganizing decks, moving cards between decks, or bulk deletion.

## Entering Multi-Select Mode

### Method 1: From Card Menu

1. **Tap a card** in Cards List
2. **Select "Select Card"** from menu
3. **Multi-select mode activates**
4. **Card is highlighted** (usually with checkmark or different background)

### Method 2: Long-Press (if supported)

1. **Long-press a card**
2. **Hold until multi-select mode activates**
3. **Card is selected**

Once in multi-select mode, the interface changes to show bulk action options.

---

## Selecting Cards

### Add to Selection

**While in multi-select mode:**
- **Tap any card** to select it
- **Checkmark appears** (or card highlights)
- **Tap again** to deselect it

### Select Multiple Cards

**Quickly select many cards:**
1. Enter multi-select mode (tap first card)
2. **Tap each card** you want to include
3. All selected cards show checkmarks/highlighting
4. **Selection counter** shows how many selected (e.g., "5 cards selected")

### Select All (if available)

Some versions may offer:
- **"Select All" button** - Selects every card in current view
- Useful for bulk operations on entire deck

## Available Bulk Operations

### Cut Cards

**Move cards to a different deck or position:**

1. **Select cards** you want to move
2. **Tap "Cut" action** (‚úÇÔ∏è icon or menu)
3. **Cards enter "cut" state** (may dim or show indicator)
4. **Navigate to destination**:
   - Different deck
   - Different position in same deck
5. **Tap "Paste" action**
6. **Cards move** to new location

**Use cases:**
- Move vocabulary cards to a different deck
- Reorganize deck structure
- Consolidate related cards

---

### Paste Cards

**After cutting cards:**

**Paste Before:**
1. Navigate to destination
2. **Tap a card** where you want to insert
3. **Select "Paste Before"**
4. Cut cards appear before the selected card

**Paste After:**
1. Navigate to destination
2. **Tap a card** where you want to insert
3. **Select "Paste After"**
4. Cut cards appear after the selected card

**Paste at End:**
1. Navigate to destination deck
2. **Select "Paste"** from deck menu
3. Cut cards append to end of deck

---

### Delete Selected Cards

**Bulk deletion:**

1. **Select cards** to delete
2. **Tap "Delete" action** (üóëÔ∏è icon or menu)
3. **Confirm deletion** (usually shows count: "Delete 7 cards?")
4. **All selected cards removed**

**‚ö†Ô∏è Warning:**
- Usually **no undo** for bulk deletion
- Double-check selection before confirming
- Consider backing up first

See [Deleting Cards](/docs/cards-list/deleting/) for more details.

---

### Copy Cards (if available)

**Duplicate cards:**

1. **Select cards** to copy
2. **Tap "Copy" action**
3. **Navigate to destination**
4. **Tap "Paste"**
5. **Duplicates appear** at destination

**Use cases:**
- Create variations of cards
- Share cards between decks
- Build related decks from common cards

**Note:** Copy feature availability may vary by version.

## Exiting Multi-Select Mode

### Deselect All

**Methods to exit:**
- **Tap "Done"** or **"X" button** (if shown)
- **Tap back button**
- **Tap empty area** (may work in some versions)

Selection clears and interface returns to normal.

### Cancel Operation

If you cut cards but changed your mind:
1. **Exit multi-select mode** without pasting
2. **Cards return to original positions**
3. **No changes made**

## Multi-Select Workflows

### Move Cards Between Decks

**Scenario:** You have a "Spanish" deck but want to split vocabulary into a separate "Spanish Vocabulary" deck.

**Steps:**
1. **Open "Spanish" deck** ‚Üí List Cards
2. **Enter multi-select mode**
3. **Select all vocabulary cards** (tap each one)
4. **Tap "Cut"**
5. **Back to home screen**
6. **Open "Spanish Vocabulary" deck** (or create it first)
7. **List Cards**
8. **Tap "Paste"** or select position and paste
9. **Vocabulary cards moved**

---

### Bulk Delete Outdated Cards

**Scenario:** Remove all cards tagged with "2024" that are no longer relevant.

**Steps:**
1. **Search for "2024"** in Cards List
2. **Enter multi-select mode**
3. **Select all search results** (or individually tap cards to delete)
4. **Tap "Delete"**
5. **Confirm deletion**
6. **All selected cards removed**

---

### Reorganize Deck Sections

**Scenario:** Move all "Chapter 5" cards to the beginning.

**Steps:**
1. **Search for "Chapter 5"**
2. **Enter multi-select mode**
3. **Select all results**
4. **Tap "Cut"**
5. **Clear search** (to see full deck)
6. **Tap card 1** (or wherever you want them)
7. **Paste Before**
8. **Chapter 5 cards now at top**

---

### Consolidate Duplicates

**Scenario:** You have duplicate cards across two decks.

**Steps:**
1. **Open first deck** ‚Üí List Cards
2. **Find duplicates** (search or visual scan)
3. **Select duplicates** (multi-select mode)
4. **Delete** (removing from first deck)
5. **Keep originals in second deck**

**Alternative:** Keep both versions if they're slightly different (different phrasing, etc.)

## Tips for Efficient Multi-Select

### Use Search First

**Before multi-selecting:**
1. **Search to filter** cards you want to select
2. **Enter multi-select mode**
3. **Select from filtered results** (faster than scrolling through entire deck)

### Check Selection Count

**Selection counter** (e.g., "12 cards selected") helps verify:
- You selected the right number
- You didn't miss any cards
- You didn't over-select

### Preview Before Pasting

**After cutting but before pasting:**
- **Navigate to destination**
- **Verify it's the right location**
- **Then paste**
- Can't undo paste easily

### Backup Before Major Moves

Before moving 50+ cards:
1. **Export database** as backup
2. **Perform multi-select operations**
3. **If something goes wrong:** Restore from backup

## Limitations

### Can't Partially Edit

Multi-select is for moving/deleting, not editing:
- ‚ùå Can't bulk-edit selected cards' content
- ‚ùå Can't apply formatting to multiple cards at once
- ‚úÖ For bulk edits, use [Excel/CSV Export](/docs/sync-export-import/export-cards/), edit in Excel, re-import

### Selection Doesn't Persist

**If you exit multi-select mode:**
- Selection clears
- Have to re-select cards if you return
- Complete your operation before exiting

### Cut/Paste is Deck-Specific

**Can only cut/paste within or between decks:**
- ‚ùå Can't cut from one GoCards database and paste to another device
- ‚úÖ Use [Export/Import Database](/docs/home/database-backup/) for cross-device transfers

## Multi-Select and File Sync

### Syncing After Multi-Select Operations

**If auto-sync enabled:**
- Moved cards update in Excel file on next sync
- Rows reorder to match GoCards
- Deleted cards removed from Excel

**If manual sync:**
- Changes don't sync until you trigger sync
- Excel file unchanged until then

### Conflicts

**Scenario:** You cut/paste cards in GoCards, but Excel file was edited with different card orders.

**Resolution:**
- Newest change wins (based on last modified timestamp)
- GoCards compares modification dates
- Usually GoCards changes persist if done after Excel edits

See [Sync Cards](/docs/sync-export-import/sync-cards/) for conflict details.
