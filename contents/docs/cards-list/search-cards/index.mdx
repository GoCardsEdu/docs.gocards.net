## Searching Cards

### How to Search

1. **Tap the 🔍 search icon** (top-right)
2. **Type your query** in the search field
3. **Results filter instantly** as you type
4. **Tap X** to clear search

### Search Features

- **Searches both** term and definition
- **Case-insensitive** - "DNA" matches "dna"
- **Partial matches** - "proto" matches "protocol" and "protein"
- **Real-time** - Updates as you type
- **Highlights matches**
