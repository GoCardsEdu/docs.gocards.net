## How to Import Cards

1. **Go to home screen** (Recent Decks or All Decks tab)
2. **Tap "⋮" menu** (top-right)
3. **Select "Import Excel"** or **"Import CSV"**
4. **Select the file** (Downloads, Google Drive, etc.)
5. **New deck is created** with the filename as deck name

**Before importing:** Make sure your file follows the correct format. See the [File Format Guide](/en/sync-export-import/file-format/) for specifications.

## When to Use Import vs. Sync

| Feature | Import | Sync |
|---------|--------|------|
| **Creates new deck** | ✅ Yes | ❌ No (updates existing) |
| **Preserves learning progress** | ❌ No | ✅ Yes |
| **Bi-directional updates** | ❌ No | ✅ Yes |
