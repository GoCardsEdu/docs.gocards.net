## Understanding Review Intervals

The interval shown below each button is **when you'll see this card again**.

### How Intervals Work

**New cards** (never studied):
- Again: 2 days
- Quick: 5 minutes
- Hard: 3 days
- Easy: 5 days

**Previously studied cards:**
- Intervals adapt based on your history
- More "Easy" clicks = longer intervals
- "Again" clicks = shorter intervals
- The algorithm balances memory retention with study efficiency

### Why These Intervals?

GoCards uses **spaced repetition** based on memory research:
- Review just before you're about to forget
- Longer intervals strengthen long-term memory
- Too frequent = waste of time
- Too infrequent = you'll forget
