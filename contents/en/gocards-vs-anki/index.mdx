## Spaced Repetition Algorithm

### GoCards

GoCards uses a straightforward multiplier system:

- **Hard:** next interval = current interval Ã— 1.5
- **Easy:** next interval = current interval Ã— 2

**Example (Easy):** 5 days â†’ 10 â†’ 20 â†’ 40 â†’ 80 â†’ 160 days

The progression is predictable. Every card follows the same multiplier regardless of history.

### AnkiDroid

AnkiDroid uses the **SM-2 algorithm** (modified) with a per-card **ease factor**:

- Each card starts with an ease factor of 2.5
- Pressing **"Hard"** decreases the ease factor
- Pressing **"Easy"** increases it
- The ease factor adjusts how aggressively intervals grow

This means two cards with the same interval can grow at different rates depending on how often you've struggled with them.

### Quizlet

Quizlet uses a proprietary scheduling algorithm:

- Cards you mark as **"Don't Know"** appear more frequently
- Cards you mark as **"Know"** appear less often
- Full spaced repetition scheduling requires **Quizlet Plus** (paid subscription)
- The algorithm is not transparent â€” you can't see or predict intervals

### Which is Better?

- **GoCards** is simpler and more predictable â€” you always know how intervals will grow
- **AnkiDroid** adapts per-card, which can be more efficient for large collections with varying difficulty
- **Quizlet** is the easiest to use but offers the least control over scheduling

## Who Should Choose Which?

**Choose GoCards if you:**
- Want to edit flashcards in Excel or Google Sheets
- Need bulk editing with find-replace across hundreds of cards
- Prefer gesture-based study (swipe, pinch, drag)
- Study programming and want built-in code highlighting
- Want to embed YouTube videos in cards
- Prefer a simpler, more predictable spaced repetition system

**Choose AnkiDroid if you:**
- Want cloze deletions or custom card templates
- Rely on shared deck libraries
- Need audio playback on cards
- Prefer a per-card adaptive algorithm
- Use Anki Desktop and want cross-platform sync via AnkiWeb

**Choose Quizlet if you:**
- Want the easiest setup with no learning curve
- Prefer multiple study modes (Learn, Test, Match)
- Need text-to-speech pronunciation
- Want to quickly find and use sets made by other students
- Don't need advanced formatting or code support
